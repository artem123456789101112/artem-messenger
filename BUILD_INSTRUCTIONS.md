## 🚀 ЗАПУСК СБОРКИ:

1. Откройте: https://github.com/artem123456789101112/artem-messenger/actions/workflows/android.yml
2. Нажмите "Run workflow"
3. Ждите 1-2 часа
4. Скачайте APK из Artifacts

## 📱 ТЕСТИРОВАНИЕ APK:

1. Скачайте APK с GitHub
2. Включите "Неизвестные источники" в настройках телефона
3. Установите APK
4. Запустите приложение

## 🔧 ЕСЛИ СБОРКА УПАДЕТ:

1. Проверьте логи в разделе Artifacts → Build-Logs
2. Если ошибка Cython - убедитесь что установлен cython
3. Если ошибка SDK - установите Android SDK вручную
